Sintaxis básica de CSS

Una regla CSS tiene la siguiente estructura:

selector {
  propiedad: valor;
  propiedad: valor;
}

Explicación:

  • selector: Indica a qué elemento(s) HTML se aplicará la regla.
  • propiedad: Es el atributo que se quiere modificar (color, tamaño, etc.).
  • valor: Es el valor que se asigna a la propiedad.

Ejemplo:

p {
  color: blue;
  font-size: 16px;
}

Este código CSS hará que todos los párrafos (<p>) tengan texto azul y un tamaño de fuente de 16 píxeles.

Selectores

Los selectores se utilizan para "seleccionar" los elementos HTML a los que se aplicarán las reglas CSS.

Tipos de selectores:

  • Selector de etiqueta: Selecciona elementos por su nombre de etiqueta (p, h1, div, etc.).
    p { color: green; }
  • Selector de clase: Selecciona elementos por su atributo de clase (.mi-clase).
    .destacado { font-weight: bold; }
  • Selector de ID: Selecciona un elemento por su atributo de ID (#mi-elemento).
    #menu { background-color: #f0f0f0; }
  • Selector universal: Selecciona todos los elementos (*).
    * { box-sizing: border-box; }
  • Selector de atributo: Selecciona elementos en función de sus atributos ([type="text"]).
    input[type="text"] { background-color: light yellow; }
  • Selectores descendientes: Selecciona elementos que son descendientes de otro elemento (div p).
    div p { font-size: 14px; }
  • Selectores hijos: Selecciona elementos que son hijos directos de otro elemento (div > p).
    div > p { color: red; }
  • Selectores adyacentes: Selecciona un elemento que está inmediatamente después de otro elemento (h1 + p).
    h1 + p { margin-top: 0; }
  • Pseudo-clases: Selecciona elementos en función de su estado (:hover, :active, :focus).
    a:hover { text-decoration: underline; }
  • Pseudo-elementos: Selecciona una parte específica de un elemento (::before, ::after, ::first-line).
    p::first-letter { font-size: 2em; }

Unidades

Las unidades se utilizan para especificar valores para las propiedades CSS.

Tipos de unidades:

  • Absolutas:
    • px (píxeles): Unidad de medida fija.
      p { font-size: 16px; }
    • cm (centímetros)
      div { width: 10cm; }
    • mm (milímetros)
      img { border-width: 2mm; }
    • in (pulgadas)
      section { margin: 1in; }
    • pt (puntos)
      h1 { font-size: 24pt; }
    • pc (picas)
      .box { line-height: 1.5pc; }
    • Relativas:
      • % (porcentaje): Relativo al elemento padre.
        div { width: 50%; }
      • em: Relativo al tamaño de fuente del elemento.
        p { margin-bottom: 1em; }
      • rem: Relativo al tamaño de fuente del elemento raíz (html).
        h1 { font-size: 2rem; }
      • vw (viewport width): Relativo al ancho de la ventana gráfica.
        .container { width: 80vw; }
      • vh (viewport height): Relativo al alto de la ventana gráfica.
        .hero { height: 100vh; }

    Ejemplo combinado:

    .box {
      width: 200px; /* Píxeles */
      height: 100px; /* Píxeles */
      padding: 10px; /* Píxeles */
      margin: 1em; /* em */
      border: 2px solid black; /* Píxeles */
    }

Colores

Hay varias formas de especificar colores en CSS:

  • Nombres de colores: red, blue, green, black, white, etc.
    p { color: red; }
  • Códigos hexadecimales: #RRGGBB (ej. #FF0000 para rojo).
    h1 { color: #0000FF; } /* Azul */
  • RGB: rgb(rojo, verde, azul) (ej. rgb(255, 0, 0) para rojo).
    div { background-color: rgb(0, 255, 0); } /* Verde */
  • RGBA: rgba(rojo, verde, azul, alfa) (ej. rgba(255, 0, 0, 0.5) para rojo semitransparente).
    .overlay { background-color: rgba(0, 0, 0, 0.5); } /* Negro semitransparente */
  • HSL: hsl(tono, saturación, luminosidad) (ej. hsl(0, 100%, 50%) para rojo).
    a { color: hsl(120, 100%, 50%); } /* Verde */
  • HSLA: hsla(tono, saturación, luminosidad, alfa) (ej. hsla(0, 100%, 50%, 0.5) para rojo semitransparente).
    .highlight { background-color: hsla(60, 100%, 50%, 0.3); } /* Amarillo semitransparente */

Modelo de caja

Cada elemento HTML se representa como una caja rectangular. El modelo de caja define las siguientes partes:

  • Contenido: El contenido real del elemento (texto, imágenes, etc.).
  • Padding: El espacio entre el contenido y el borde.
  • Borde: El borde que rodea el padding y el contenido.
  • Margen: El espacio entre el borde y los elementos adyacentes.

Propiedades:

  • width y height: Definen el ancho y alto del contenido.
    div { width: 200px; height: 100px; }
  • padding: Define el padding.
    p { padding: 20px; }
  • margin: Define el margen.
    h1 { margin: 30px; }
  • box-sizing: Define cómo se calcula el ancho y alto total del elemento (content-box o border-box).
    * { box-sizing: border-box; }

Display

La propiedad display controla cómo se comporta un elemento en el diseño de la página.

Valores comunes:

  • block: El elemento ocupa todo el ancho disponible y comienza en una nueva línea.
    div { display: block; }
  • inline: El elemento solo ocupa el espacio necesario y no comienza en una nueva línea.
    span { display: inline; }
  • inline-block: El elemento se comporta como inline, pero puede tener ancho y alto definidos.
    button { display: inline-block; }
  • none: El elemento no se muestra en la página.
    .hidden { display: none; }
  • flex: El elemento se convierte en un contenedor flex.
    .container { display: flex; }
  • grid: El elemento se convierte en un contenedor grid.
    .grid { display: grid; }
  • table, table-row, table-cell: El elemento se comporta como una tabla, fila o celda de tabla.
    div { display: table; }

Posicionamiento

La propiedad position controla cómo se posiciona un elemento en la página.

Valores comunes:

  • static: Posicionamiento por defecto. El elemento se coloca en el flujo normal del documento.
  • relative: El elemento se posiciona en relación a su posición normal en el flujo del documento.
    .box { position: relative; top: 20px; left: 30px; }
  • absolute: El elemento se posiciona en relación a su ancestro posicionado más cercano. Si no hay ninguno, se posiciona en relación al documento.
    .modal { position: absolute; top: 50%; left: 50%; transform: translate(-50%, -50%); }
  • fixed: El elemento se posiciona en relación a la ventana gráfica. Permanece en la misma posición aunque se desplace la página.
    .fixed-header { position: fixed; top: 0; left: 0; width: 100%; }
  • sticky: El elemento se comporta como relative hasta que alcanza un cierto umbral, entonces se comporta como fixed.
    .sticky-nav { position: sticky; top: 0; }

Propiedades relacionadas:

  • top, right, bottom, left: Definen la posición del elemento en relación a su contenedor.
  • z-index: Define el orden de apilamiento de los elementos. Un valor más alto se muestra encima de los elementos con valores más bajos.
    .top-layer { z-index: 10; }

Float

La propiedad float permite que un elemento flote a la izquierda o derecha de su contenedor, permitiendo que otros elementos lo rodeen.

Valores comunes:

  • left: El elemento flota a la izquierda.
    img { float: left; margin-right: 10px; }
  • right: El elemento flota a la derecha.
    .sidebar { float: right; width: 30%; }
  • none: El elemento no flota.

Para evitar que un elemento sea rodeado por elementos flotantes, se puede utilizar la propiedad clear:

  • left: Evita que elementos flotantes a la izquierda lo rodeen.
  • right: Evita que elementos flotantes a la derecha lo rodeen.
  • both: Evita que elementos flotantes a ambos lados lo rodeen.
    .clear { clear: both; }

Flexbox

Flexbox es un módulo de CSS que proporciona una forma flexible de diseñar elementos en una dimensión (horizontal o vertical).

Propiedades del contenedor flex:

  • display: flex;: Convierte el elemento en un contenedor flex.
  • flex-direction: Define la dirección de los elementos flex (row, column, row-reverse, column-reverse).
    .container { flex-direction: column; }
  • justify-content: Define la alineación de los elementos flex en el eje principal (flex-start, flex-end, center, space-between, space-around).
    .container { justify-content: center; }
  • align-items: Define la alineación de los elementos flex en el eje transversal (flex-start, flex-end, center, stretch, baseline).
    .container { align-items: center; }
  • flex-wrap: Define si los elementos flex deben ajustarse a una sola línea o no (nowrap, wrap, wrap-reverse).
    .container { flex-wrap: wrap; }

Propiedades de los elementos flex:

  • order: Define el orden de los elementos flex.
    .item-2 { order: 1; }
  • flex-grow: Define cuánto espacio debe ocupar un elemento flex en relación a los demás.
    .item-1 { flex-grow: 2; }
  • flex-shrink: Define cuánto debe encogerse un elemento flex en relación a los demás.
    .item-3 { flex-shrink: 0; }
  • flex-basis: Define el tamaño inicial de un elemento flex.
    .item-2 { flex-basis: 150px; }
  • align-self: Permite anular la alineación definida por align-items para un elemento flex específico.
    .item-3 { align-self: flex-end; }

Grid

Grid es un módulo de CSS que proporciona una forma de diseñar elementos en dos dimensiones (filas y columnas).

Propiedades del contenedor grid:

  • display: grid;: Convierte el elemento en un contenedor grid.
  • grid-template-columns: Define las columnas del grid.
    .grid { grid-template-columns: 1fr 2fr 1fr; }
  • grid-template-rows: Define las filas del grid.
    .grid { grid-template-rows: 100px 200px; }
  • grid-gap: Define el espacio entre las filas y columnas.
    .grid { grid-gap: 10px; }
  • justify-content: Define la alineación de la grid en el eje horizontal.
    .grid { justify-content: center; }
  • align-content: Define la alineación de la grid en el eje vertical.
    .grid { align-content: space-around; }

Propiedades de los elementos grid:

  • grid-column-start, grid-column-end: Definen la posición de un elemento en las columnas.
    .item-1 { grid-column-start: 1; grid-column-end: 3; }
  • grid-row-start, grid-row-end: Definen la posición de un elemento en las filas.
    .item-2 { grid-row-start: 2; grid-row-end: 4; }

Media Queries

Las Media Queries permiten aplicar diferentes estilos CSS en función de las características del dispositivo (ancho de pantalla, orientación, etc.).

Sintaxis:

@media (condición) {
  /* Reglas CSS */
}

Ejemplos de condiciones:

  • (max-width: 768px): Se aplica cuando el ancho de pantalla es menor o igual a 768px.
  • (orientation: landscape): Se aplica cuando la orientación del dispositivo es horizontal.
  • (min-resolution: 2dppx): Se aplica cuando la resolución del dispositivo es al menos 2 puntos por píxel.

Ejemplo:

@media (max-width: 768px) {
  .sidebar {
    display: none;
  }

Animaciones

CSS permite crear animaciones mediante la propiedad animation.

Primero, se define una animación con @keyframes:

@keyframes miAnimacion {
  from {
    background-color: red;
  }
  to {
    background-color: blue;
  }
}

Luego, se aplica la animación a un elemento:

.box {
  width: 100px;
  height: 100px;
  background-color: red;
  animation-name: miAnimacion;
  animation-duration: 5s;
}

Propiedades de animación:

  • animation-name: Nombre de la animación.
  • animation-duration: Duración de la animación.
  • animation-timing-function: Función de temporización (ease, linear, ease-in, ease-out, ease-in-out).
  • animation-delay: Retraso antes de iniciar la animación.
  • animation-iteration-count: Número de veces que se repite la animación (infinite para repetir indefinidamente).
  • animation-direction: Dirección de la animación (normal, reverse, alternate, alternate-reverse).
  • animation-fill-mode: Define qué valores se aplican antes y después de la animación (none, forwards, backwards, both).
  • animation-play-state: Estado de la animación (running, paused).
  • animation: Propiedad abreviada para definir todas las propiedades de animación.

Transiciones

Las transiciones permiten animar cambios en las propiedades de un elemento de forma suave.

Ejemplo:

.box {
  width: 100px;
  height: 100px;
  background-color: red;
  transition: background-color 2s ease;
}
.box:hover {
  background-color: blue;
}

Propiedades de transición:

  • transition-property: Propiedad que se animará.
  • transition-duration: Duración de la transición.
  • transition-timing-function: Función de temporización.
  • transition-delay: Retraso antes de iniciar la transición.
  • transition: Propiedad abreviada para definir todas las propiedades de transición.

Transformaciones

Las transformaciones permiten modificar la forma y posición de un elemento.

Funciones de transformación:

  • translate(x, y): Mueve el elemento en el eje x e y.
  • rotate(ángulo): Rota el elemento.
  • scale(x, y): Escala el elemento en el eje x e y.
  • skew(x-ángulo, y-ángulo): Inclina el elemento en el eje x e y.
  • matrix(a, b, c, d, e, f): Define una transformación matricial.

Ejemplo:

.box {
  width: 100px;
  height: 100px;
  background-color: red;
  transform: rotate(45deg);
}
