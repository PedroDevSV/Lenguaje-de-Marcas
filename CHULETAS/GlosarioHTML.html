Estructura básica de un documento HTML

Un documento HTML tiene la siguiente estructura básica:

<!DOCTYPE html>
<html lang="es">
<head>
  <meta charset="UTF-8">
  <title>Título de la página</title>
</head>
<body>
  <h1>Esto es un encabezado</h1>
  <p>Esto es un párrafo.</p>
</body>
</html>

Explicación:

  • <!DOCTYPE html>: Define el tipo de documento como HTML5.
  • <html lang="es">: Etiqueta raíz del documento. El atributo lang especifica el idioma.
  • <head>: Contiene información sobre la página, como el título.
  • <title>: Define el título que aparece en la pestaña del navegador.
  • <body>: Contiene el contenido visible de la página.
  • <h1>: Define un encabezado de nivel 1.
  • <p>: Define un párrafo.

Atributos

Los atributos proporcionan información adicional sobre las etiquetas HTML.

Ejemplos:

<a href="https://www.ejemplo.com" target="_blank">Abrir enlace en una nueva pestaña</a>
<img src="imagen.jpg" alt="Descripción de la imagen" width="300"><
<p style="color: blue;">Párrafo con texto azul</p>

Explicación:

  • target="_blank": Abre el enlace en una nueva pestaña del navegador.
  • width="300": Define el ancho de la imagen en píxeles.
  • style="color: blue;": Define el color del texto como azul.

Aquí tienes una tabla con algunos atributos comunes:

Atributo Descripción Ejemplo
href Especifica la URL a la que apunta un enlace. <a href="https://www.ejemplo.com">
src Especifica la URL de una imagen o recurso externo. <img src="imagen.jpg">
alt Especifica un texto alternativo para una imagen. <img src="imagen.jpg" alt="Descripción de la imagen">
width Especifica el ancho de un elemento. <img src="imagen.jpg" width="300">
height Especifica la altura de un elemento. <img src="imagen.jpg" height="200">
style Especifica estilos en línea para un elemento. <p style="color: blue;">
class Especifica una o más clases para un elemento. <div class="mi-clase">
id Especifica un identificador único para un elemento. <p id="mi-parrafo">
title Especifica un texto informativo que se muestra al pasar el cursor sobre el elemento. <a href="#" title="Más información">
disabled Deshabilita un elemento de formulario. <button disabled>

Ejemplos:

Tablas

Las tablas se utilizan para organizar datos en filas y columnas.

<table>
  <thead>
    <tr>
      <th>Encabezado 1</th>
      <th>Encabezado 2</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>Dato 1</td>
      <td>Dato 2</td>
    </tr>
    <tr>
      <td>Dato 3</td>
      <td>Dato 4</td>
    </tr>
  </tbody>
</table>

Explicación:

  • <table>: Define la tabla.
  • <thead>: Define el encabezado de la tabla.
  • <tbody>: Define el cuerpo de la tabla.
  • <tr>: Define una fila de la tabla.
  • <th>: Define una celda de encabezado.
  • <td>: Define una celda de datos.

Atributos adicionales:

  • border: Define un borde para la tabla.
  • cellpadding: Define el espacio entre el contenido de una celda y su borde.
  • cellspacing: Define el espacio entre las celdas.
  • colspan: Define el número de columnas que debe abarcar una celda.
  • rowspan: Define el número de filas que debe abarcar una celda.

Formularios

Los formularios se utilizan para recopilar información del usuario.

<form action="/procesar_datos" method="post">
  <label for="nombre">Nombre:</label>
  <input type="text" id="nombre" name="nombre">
  <br>
  <label for="email">Email:</label>
  <input type="email" id="email" name="email">
  <br>
  <input type="submit" value="Enviar">
</form>

Explicación:

  • <form>: Define el formulario.
  • action: Especifica la URL a la que se enviarán los datos del formulario.
  • method: Especifica el método HTTP para enviar los datos (get o post).
  • <label>: Define una etiqueta para un elemento de formulario.
  • <input>: Define un campo de entrada.
  • type: Especifica el tipo de campo de entrada (texto, email, contraseña, etc.).
  • <input type="submit">: Define un botón de envío.

Tipos de input

Aquí tienes una tabla con algunos tipos de input comunes:

Tipo Descripción Ejemplo
text Campo de texto de una sola línea. <input type="text" name="nombre">
password Campo de contraseña que oculta los caracteres introducidos. <input type="password" name="contraseña">
email Campo de correo electrónico que valida el formato. <input type="email" name="email">
number Campo numérico que solo permite introducir números. <input type="number" name="edad">
date Campo de fecha que permite seleccionar una fecha. <input type="date" name="fecha_nacimiento">
checkbox Casilla de verificación que permite seleccionar una o varias opciones. <input type="checkbox" name="intereses" value="deportes"> Deportes
radio Botón de opción que permite seleccionar una sola opción de un grupo. <input type="radio" name="genero" value="masculino"> Masculino
file Campo para seleccionar un archivo. <input type="file" name="archivo">
submit Botón para enviar el formulario. <input type="submit" value="Enviar">
reset Botón para restablecer los valores del formulario. <input type="reset" value="Restablecer">
button Botón genérico que no tiene una acción predefinida. <input type="button" value="Haz clic">
hidden Campo oculto que no se muestra al usuario. <input type="hidden" name="id" value="123">

Otros elementos de formulario:

  • <textarea>: Define un área de texto de varias líneas.
  • <select>: Define una lista desplegable.
  • <option>: Define una opción en una lista desplegable.
  • <fieldset>: Agrupa elementos de formulario.
  • <legend>: Define una leyenda para un <fieldset>.

Multimedia

HTML permite incrustar contenido multimedia como audio y video.

Audio

<audio controls>
  <source src="audio.mp3" type="audio/mpeg">
  <source src="audio.ogg" type="audio/ogg">
<p>Tu navegador no soporta el elemento audio.</p>
</audio>

Explicación:

  • <audio>: Define el elemento de audio.
  • controls: Muestra los controles de reproducción (reproducir, pausar, volumen, etc.).
  • <source>: Especifica diferentes fuentes de audio para compatibilidad con diferentes navegadores.
  • type: Especifica el tipo MIME del archivo de audio.

Video

<video width="640" height="360" controls>
  <source src="video.mp4" type="video/mp4">
  <source src="video.webm" type="video/webm">
<p>Tu navegador no soporta el elemento video.</p>
</video>

Explicación:

  • <video>: Define el elemento de video.
  • width y height: Especifican el ancho y alto del reproductor de video.
  • controls: Muestra los controles de reproducción.
  • <source>: Especifica diferentes fuentes de video para compatibilidad.
  • type: Especifica el tipo MIME del archivo de video.

Semántica

HTML5 introduce nuevas etiquetas semánticas que dan significado al contenido.

Ejemplos:

<header>...</header>
<nav>...</nav>
<main>...</main>
<article>...</article>
<aside>...</aside>
<footer>...</footer>

Explicación:

  • <header>: Define el encabezado de una página o sección.
  • <nav>: Define un bloque de navegación.
  • <main>: Define el contenido principal de una página.
  • <article>: Define un artículo independiente.
  • <aside>: Define contenido relacionado con el contenido principal (como una barra lateral).
  • <footer>: Define el pie de página de una página o sección.

Utilizar etiquetas semánticas mejora la accesibilidad y el SEO de tu sitio web.

Metadatos

Los metadatos proporcionan información sobre el documento HTML. Se incluyen en la sección <head> y no se muestran en la página.

Ejemplos:

<meta charset="UTF-8">
<meta name="description" content="Descripción de la página">
<meta name="keywords" content="palabra clave 1, palabra clave 2">
<meta name="author" content="Autor de la página">
<meta name="viewport" content="width=device-width, initial-scale=1.0">

Explicación:

  • <meta charset="UTF-8">: Define la codificación de caracteres del documento.
  • <meta name="description" content="...">: Define una descripción de la página para los motores de búsqueda.
  • <meta name="keywords" content="...">: Define palabras clave relevantes para la página.
  • <meta name="author" content="...">: Define el autor de la página.
  • <meta name="viewport" content="...">: Configura cómo se muestra la página en diferentes dispositivos.

Iframes

Los iframes permiten incrustar otra página web dentro de la página actual.

Ejemplo:

<iframe src="https://www.ejemplo.com" width="800" height="600"></iframe>

Explicación:

  • <iframe>: Define el iframe.
  • src: Especifica la URL de la página web que se va a incrustar.
  • width y height: Especifican el ancho y alto del iframe.

Atributos adicionales:

  • title: Especifica un título para el iframe.
  • allowfullscreen: Permite que el iframe se muestre en pantalla completa.
  • sandbox: Aplica restricciones de seguridad al iframe.
